# Turbo-Game-
An excellent clicker game for PC. Use the manual download option, because automatic installation may place the files in Program Files due to a known bug. Please read the README for proper installation. The game is only available in Russian, as there is no English translation yet. Thanks for understanding! Future updates will include an English ver.
in order to download the game you need to
1)download installer
2)select manual installation
3)create a folder in which you want to put the game
4)whether to make a shortcut or not is up to you.
5)Launch the game and enjoy
